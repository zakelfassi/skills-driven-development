# Packs

A **pack** is a themed, importable set of skills that travel together — forged around one methodology, event, or workflow family rather than one project.

Packs are just directories of spec-compliant skills plus a README. There is no special tooling: install a pack by copying its skill directories into a colony (project `skills/` or global `~/.skdd/skills/`), registering them in `.skills-registry.md`, and running `skdd link` (or `skdd link -g`) to refresh harness mirrors.
