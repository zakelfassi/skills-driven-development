# Code of Conduct

This project adopts the **Contributor Covenant v2.1** as its code of conduct.

The canonical text lives at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/> and is incorporated here by reference. A translated version in your language is probably available at <https://www.contributor-covenant.org/translations/>.

In one sentence: **be excellent to each other, and assume good faith.** Harassment, personal attacks, and demographic slurs are not tolerated in issues, pull requests, discussions, or any other project space.

## Scope

This Code applies to all project spaces (this repository, any associated Discussions, any Discord / Matrix / Slack communities, and public events where a maintainer represents the project) and to one-on-one communications where someone is acting in their capacity as a SkDD contributor.

## Enforcement

Reports go to the maintainer directly at `zakelfassi+conduct@gmail.com`.

Expected response:

- **Acknowledgement** within 72 hours
- **Initial assessment** within 7 days
- **Resolution** (warning, temporary ban, permanent ban, or "no action") communicated to the reporter

Maintainers follow the Contributor Covenant's Enforcement Ladder: Correction → Warning → Temporary Ban → Permanent Ban. Which step applies depends on severity, context, and repeat behavior.

Reports will be handled discreetly. The reporter's identity will not be shared with the reported party unless the reporter explicitly consents.
